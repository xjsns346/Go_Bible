1. 有关文件路径处理的常用函数:

| 函数                          | 作用                   |
| :---| :---|
| `filepath.Dir(path)`        | 获取路径的父目录             |
| `filepath.Base(path)`       | 获取路径的最后一部分（文件名或最后目录） |
| `filepath.Ext(path)`        | 获取文件扩展名              |
| `filepath.Join(a, b, c...)` | 拼接路径，自动加分隔符          |



2. 与输出有关的函数

| 函数            | 功能                 | 输出位置           |
| ------------- | ------------------ | -------------- |
| `fmt.Printf`  | 格式化并打印             | 控制台（标准输出）      |
| `fmt.Sprintf` | 格式化并返回字符串          | 返回字符串变量（不打印）   |
| `fmt.Fprintf` | 格式化并写入指定 io.Writer | 可输出到文件、网络、缓冲区等 |

# 函数具体定义

## fmt.Sprintf函数具体信息  

| 项目   | 说明                                             |
| ---- | ---------------------------------------------- |
| 函数原型 | `func Sprintf(format string, a ...any) string` |
| 作用   | 格式化字符串并返回                                      |
| 输出位置 | 内存中（非控制台）                                      |
| 常见用途 | 构造字符串、实现 `String()` 方法、日志格式化                   |
| 注意事项 | 需要手动打印或返回结果，否则不会显示                             |
| 对比函数 | `Printf`（打印） / `Fprintf`（输出到 Writer）           |
