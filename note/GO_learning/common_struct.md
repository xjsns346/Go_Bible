# GO语言中常见结构体简介

## scanner

| 名称                   | 类型              | 作用                                   |
| -------------------- | --------------- | ------------------------------------ |
| `Scanner`            | 结构体             | 封装了缓冲读取与分词逻辑                         |
| `r`                  | `io.Reader`     | 输入源，如字符串、文件等                         |
| `SplitFunc`          | 函数类型            | 定义如何分割数据                             |
| `Scan()`             | 方法              | 逐步扫描下一个 token                        |
| `Text()` / `Bytes()` | 方法              | 获取当前 token 内容                        |
| `Err()`              | 方法              | 返回扫描过程中的错误                           |
| 默认分割方式               | 行 (`ScanLines`) | 可切换为词 (`ScanWords`)、字节 (`ScanBytes`) |
